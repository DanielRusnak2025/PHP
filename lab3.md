
## Задание 1. Расписание на основе дня недели (`date()`)

> Нужно вывести таблицу с расписанием, где у каждого сотрудника свои рабочие дни.

```
<?php

// Получаем текущий день недели (1 = пн, 2 = вт, ..., 7 = вс)
$day = (int) date('N');

// Определяем график для John Styles
// Рабочие дни: понедельник (1), среда (3), пятница (5)
if ($day === 1 || $day === 3 || $day === 5) {
    $johnSchedule = "8:00 - 12:00";
} else {
    $johnSchedule = "Нерабочий день";
}

// Определяем график для Jane Doe
// Рабочие дни: вторник (2), четверг (4), суббота (6)
if ($day === 2 || $day === 4 || $day === 6) {
    $janeSchedule = "12:00 - 16:00";
} else {
    $janeSchedule = "Нерабочий день";
}

// Выводим таблицу
echo "<table border='1' cellpadding='8' cellspacing='0'>";
echo "<tr>
    <th>№</th>
    <th>Фамилия Имя</th>
    <th>График работы</th>
</tr>";

echo "<tr>
    <td>1</td>
    <td>John Styles</td>
    <td>{$johnSchedule}</td>
</tr>";

echo "<tr>
    <td>2</td>
    <td>Jane Doe</td>
    <td>{$janeSchedule}</td>
</tr>";

echo "</table>";
?>
```


- `date('N')` возвращает день недели в формате ISO 8601: `1` = понедельник, `7` = воскресенье.
- Для каждого сотрудника проверяем номер дня и подставляем нужный текст в переменную.
- Потом просто выводим таблицу с этими переменными — удобно и читаемо.

## Задание 2. Циклы

### 2.1 Исходный код (`index.php`)

```
<?php

$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
    $a += 10;
    $b += 5;
}

echo "End of the loop: a = $a, b = $b";
```

---

### 2.2 Добавляем вывод промежуточных значений

> Чтобы видеть что происходит на каждом шаге, добавляем `echo` внутрь цикла.

```
<?php

$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
    $a += 10;
    $b += 5;

    // Выводим значения на каждом шаге
    echo "Шаг $i: a = $a, b = $b <br>";
}

echo "End of the loop: a = $a, b = $b";
```

**Вывод будет примерно такой:**
```
Шаг 0: a = 10, b = 5
Шаг 1: a = 20, b = 10
Шаг 2: a = 30, b = 15
Шаг 3: a = 40, b = 20
Шаг 4: a = 50, b = 25
Шаг 5: a = 60, b = 30
End of the loop: a = 60, b = 30
```

---

### 2.3 Переписываем через `while`

> `while` проверяет условие **до** выполнения тела цикла.

```
<?php

$a = 0;
$b = 0;
$i = 0; // счётчик объявляем отдельно, до цикла

while ($i <= 5) {
    $a += 10;
    $b += 5;

    echo "Шаг $i: a = $a, b = $b <br>";

    $i++; // не забываем увеличивать счётчик, иначе будет бесконечный цикл!
}

echo "End of the loop: a = $a, b = $b";
```

---

### 2.4 Переписываем через `do-while`

> `do-while` выполняет тело цикла **хотя бы один раз**, и только потом проверяет условие.

```
<?php

$a = 0;
$b = 0;
$i = 0;

do {
    $a += 10;
    $b += 5;

    echo "Шаг $i: a = $a, b = $b <br>";

    $i++;
} while ($i <= 5); // условие проверяется в конце

echo "End of the loop: a = $a, b = $b";
```


- `while` — сначала проверяет, потом выполняет. Если условие ложно с самого начала — тело не выполнится ни разу.
- `do-while` — сначала выполняет, потом проверяет. Тело выполнится хотя бы один раз, даже если условие изначально ложно.
