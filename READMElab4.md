## 1. Что такое массивы в PHP?

**Массив** в PHP — это упорядоченная структура данных, которая хранит одно или несколько значений в одной переменной. Каждое значение в массиве имеет свой **ключ** (индекс), по которому к нему можно обратиться.

PHP поддерживает три вида массивов:
- **Индексированные** — ключи задаются автоматически числами (0, 1, 2, ...);
- **Ассоциативные** — ключи задаются вручную в виде строк;
- **Многомерные** — массивы, элементами которых являются другие массивы.

## 2. Каким образом можно создать массив в PHP?

В PHP существует несколько способов создания массивов:

### Способ 1 — Краткий синтаксис `[]` (рекомендуется, PHP 5.4+)
```php
// Индексированный массив
$fruits = ['apple', 'banana', 'cherry'];

// Ассоциативный массив
$user = [
    'name' => 'Alice',
    'age'  => 25,
    'city' => 'Chisinau',
];
```

### Способ 2 — Функция `array()`
```php
// Индексированный массив
$fruits = array('apple', 'banana', 'cherry');

// Ассоциативный массив
$user = array(
    'name' => 'Alice',
    'age'  => 25,
);
```

### Способ 3 — Поэлементное добавление
```php
$colors = [];
$colors[] = 'red';    // автоматический числовой ключ
$colors[] = 'green';
$colors['favorite'] = 'blue';  // строковый ключ
```

### Способ 4 — Многомерный массив (как в лабораторной)
```php
$transactions = [
    [
        'id'     => 1,
        'amount' => 100.00,
        'date'   => '2024-01-01',
    ],
    [
        'id'     => 2,
        'amount' => 75.50,
        'date'   => '2024-02-15',
    ],
];
```

## 3. Для чего используется цикл `foreach`?

Цикл **`foreach`** предназначен для перебора всех элементов массива (или объекта, реализующего интерфейс `Iterable`). Это самый удобный способ итерации по массивам в PHP, так как он автоматически перебирает все элементы от начала до конца без необходимости вручную управлять индексами.

### Синтаксис:
```php
// Только значения
foreach ($array as $value) {
    // работаем с $value
}

// Ключ и значение
foreach ($array as $key => $value) {
    // работаем с $key и $value
}
```

### Пример из лабораторной работы:
```php
// Подсчёт общей суммы транзакций
function calculateTotalAmount(array $transactions): float
{
    $total = 0.0;
    foreach ($transactions as $t) {
        $total += $t['amount'];
    }
    return $total;
}

// Вывод таблицы в HTML
foreach ($transactions as $t) {
    echo "<tr>";
    echo "<td>" . $t['id'] . "</td>";
    echo "<td>" . $t['amount'] . "</td>";
    echo "</tr>";
}
```

### Преимущества `foreach` перед `for`:
- Не нужно знать размер массива (`count()`);
- Работает с ассоциативными массивами (строковые ключи);
- Код читается проще и понятнее;
- Не выходит за границы массива.
